Cell Balancing Design

Driving mode (BalancingOFF)

All the switches of the cell monitoring unit are open, so that load current can easily be drawn from the battery (otherwise, the resistors in parallel would just increase load uselessly).

Charging mode (BalancingON)

All the switches of the cell monitoring unit are closed except from the one with the lowest SOC. This makes sure that the cell with lowest SOC is charged up more fast, because all the charging current flows through it and it is not split with the resistor in parallel.

Stand-by mode (BalancingON)

All the switches of the cell monitoring unit are closed except from the one with the lowest SOC. This makes sure that the SOC of all the cells (except from the one with lowest SOC) is slowly reduced, so that eventually all the SOCs are balanced. This is because current flows internally through the cell and its resistor in parallel when the MOSFET is closed. No current flows out of the battery, since the contactors with load and charger are open.
